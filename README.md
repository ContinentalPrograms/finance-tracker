# Personal Finance Tracker

A beginner Python + SQL project to track income and expenses.

## Features
- Add income and expense transactions
- View all transactions
- See category-wise summary and balance

## Tech Stack
- Python 3
- SQLite (built-in sqlite3 module)

## How to Run
python3 finance_tracker.py

## Files
- finance_tracker.py - Main app
- database.py - Database logic
